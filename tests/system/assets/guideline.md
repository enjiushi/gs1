# tests/system/assets/

Asset-authored `.gs1systemtest` packs grouped by gameplay system or regression area.

## Usage
- Read this file before scanning the folder in detail.
- If you open a child folder, read its `guideline.md` first when present.
- When files or folders in this directory change, update this file in the same change.

## Contents
- `README.md`: Authoring overview and metadata format for system-test asset files.
- `ecology/`: Ecology regression assets for watering, burial, and growth behavior.
- `economy_phone/`: Economy phone/shop regression assets.
- `inventory/`: Inventory use and transfer regression assets.
- `placement_validation/`: Placement validation acceptance/rejection regression assets.
- `task_board/`: Task board reset regression assets.
