# src/content/tables/

External prototype content TOML files loaded by the gameplay DLL at startup.

## Usage
- Read this file before scanning the folder in detail.
- When files or folders in this directory change, update this file in the same change.

## Contents
- `campaign_setup.toml`: Prototype campaign setup content for starting cash, support quota per contributor, and baseline deployment items, with authored cash normalized to internal cash points at load.
- `sites.toml`: Prototype campaign/site authoring content for regional-map tile placement, objective config, support exports, nearby aura seeds, completion rewards, camp anchors, site default-weather baselines including Site 1's hotter starting heat, authored starter-plant patches such as Site 1's harvestable near-camp starter growth, initial reveal/availability flags, and per-site phone delivery fees authored in player cash but normalized to internal cash points at load.
- `phone_listings.toml`: Seeded phone-listing content for per-site buy, unlockable, and contractor offers, including buy-item stock seeded without authored item cash because those prices derive from item internal cash points, direct-purchase buff consumables that still use the normal buy-item listing path, plus internal cash-point valuation for direct-purchase unlockables that converts to player-facing cash.
- `items.toml`: Item authoring content for runtime item definitions and item-domain metadata, including consumable meter deltas that can derive consumable internal cash-point valuation from the shared player-meter cash-point tuning, optional shared `modifier_id` references that add the current flat timed-modifier premium when the referenced modifier duration is nonzero, fallback internal cash-point valuation for non-meter goods used by task/reward scoring and phone/store pricing, the full prototype seed-item roster, harvested-good items for the China-desert ten-plant set, and the current excavation-only stone merchandise balance rows.
- `excavation_tables.toml`: Excavation depth tuning and depth-scoped tier `100%` loot pools for the prototype desert stone finds, including the current `50%` any-find balance plus distinct within-tier splits for rough/careful/thorough digging.
- `plants.toml`: Plant authoring content for ecology, explicit focus tags, the authored six-meter roster-pool model (`salt`, `heat`, `wind`, `dust`, `fertility`, `output`), footprint-multiple aura and wind shelter reach, authored nearby-protection ratios over resistance-driven outward shelter, salt-derived plant salinity rehab for support foci, the straw checkerboard's steady five-real-minute constant wither behavior, and per-plant harvest thresholds/output tuning across the full prototype China-desert 10-plant roster.
- `structures.toml`: Structure authoring content for buildables, storage, crafting stations, and authored resolved-tile support channels such as wind, heat, dust, fertility, salinity, and irrigation output.
- `craft_recipes.toml`: Craft recipe authoring content for station/output/ingredient mappings plus recipe-authored craft duration and worker energy/hydration/nourishment costs, including the current buff-granting cooked drink and meal examples.
- `task_templates.toml`: Authored task template content using single target/required-count/threshold values, where `0` or `None` means the runtime must resolve that field procedurally, plus authored expected in-game hours and risk multipliers for valuation and optional selector hints for runtime-picked item/plant/recipe/structure/action values.
- `site_onboarding_task_seeds.toml`: Site-keyed onboarding task seed content that instantiates exact task instances for a specific site, with optional fixed reward-candidate assignment.
- `reward_candidates.toml`: Reward candidate content for money, deliveries, unlockable reveals, run modifiers, and reputation rewards, with authored cash grants normalized to internal cash points at load.
- `site_actions.toml`: Site action tuning content for shared action duration and worker energy/hydration/nourishment/morale costs, authored local-weather-to-meter cost coefficients, placement occupancy, and reservation/approach/movement flags, with craft duration and worker meter costs authored per recipe, the shared harvest action profile, and the authored excavate action profile loaded here.
- `modifier_presets.toml`: Unified modifier content for nearby auras, permanent run modifiers, and timed consumable modifiers, with every entry authored behind an explicit `modifier_id`, optional display text, `0` duration for permanent effects, positive `8`-hour duration blocks for timed effects, optional `timed_buff_cap_bias`, and worker action-cost modifier deltas.
- `gameplay_tuning.toml`: Shared numeric gameplay-tuning table for worker-condition, player-meter internal cash-point valuation, item buy/sell multipliers, ecology, modifier, device-support, and camp-durability factors, including worker background energy-recovery tuning, the modifier system's configurable base active timed-buff cap, the current higher base plant-pressure floor with lighter dust weighting, the shared `0.01` weather-vs-resistance growth-pressure comparison that should remain data-authored instead of hardcoded, and the ecology full-range real-minutes tuning for the linear density-change curve, currently tuned to a `5` real-minute full-range baseline.
- `technology_tiers.toml`: Faction tech-tree tier content for tier ordering and display names across the full authored `8`-tier branch.
- `reputation_unlocks.toml`: Total-reputation unlock content for one-by-one post-starter plant unlocks and future non-plant unlock types.
- `technology_nodes.toml`: Technology node content for one faction-owned base tech plus `2` enhancement choices per tier, authored faction reputation requirements plus internal cash-point valuation that converts to player-facing cash, reputation-scaled effect parameters, optional item/plant/structure/recipe grant payloads, entry kinds, and placeholder labels/descriptions.
- `initial_unlocked_plants.toml`: Starter-plant content for the two plants available from campaign start before later reputation unlocks.
