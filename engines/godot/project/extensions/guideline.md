# engines/godot/project/extensions/

Godot runtime-extension packaging files referenced by the project shell.

## Usage
- Read this file before scanning the folder in detail.
- If you open a child folder, read its `guideline.md` first when present.
- When files or folders in this directory change, update this file in the same change.

## Contents
- `gs1_adapter/`: Thin runtime extension package metadata that points Godot at the built GS1 native adapter DLL.
